# Praktikum-web

Tasty Bites adalah sebuah website restoran yang dibuat untuk membantu para calon pelanggan untuk reservasi kursi pada restoran Tasty Bites.

Pada index.html itu menampilkan page about atau page awal dari website tersebut, berisi deskripsi dari 
restoran Tasty Bites dan juga foto foto makanan yang ada di Tasty bites.

Pada page Reservation adalah page menampilkan form untuk para calon pelanggan untuk reservasi kursi pada restoran,
selain itu page ini memerintahkan pelanggan untuk memasukkan nama,email,alasan ingin reservasi,
apakah ia bersedia untuk membayar cash atau debit,berapa jumlah kursi yang di reservasi,
ada juga paket menu nya lalu juga ada mengupload foto diri.

Pada page menu terdapat banyak hidangan hidangan yang bisa disediakan oleh restoran.

Yang terakhir ada Page Owner, Pada page ini menampilkan beberapa owner sekaligus chef
pada restoran Tasty Bites.
